# x402 v2

x402 is the HTTP 402 ("Payment Required") protocol — instant atomic USDC payments per API call, no escrow. The X402Adapter routes any `https://…` URL through this path.

## Mainnet vs sepolia

- **Mainnet** — direct buyer → seller via `@x402/fetch` + facilitator. **Zero AGIRAILS fee.** `x402_relay` is `null` on mainnet by design.
- **Sepolia** — `X402Relay` contract is still deployed for fee-splitting integration tests. Marked `deprecated` in `deployments/base-sepolia.json`. Configure `relay_address` in `X402AdapterConfig` to opt in.

## SDK surface

```typescript
await client.pay({ to: "https://api.example.com/predict", amount: "0.01" });
```

Routes to X402Adapter (priority 70), which speaks x402 v2 (EIP-3009 / Permit2 signing, CAIP-2 networks). Interoperable with any x402 v2 server (Coinbase, third-party, self-hosted).
